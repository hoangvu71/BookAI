## Technology Stack

- **Frontend**: Open WebUI interface
- **Backend**: Node.js/Express with Open WebUI integration
- **Database**: Supabase (PostgreSQL)
- **AI Model**: Google Cloud Vertex AI (Gemini 2.0 Flash)
- **Deployment**: Docker containerization
